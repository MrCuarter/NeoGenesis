
# Guía de Usuario - NeoGenesis

Bienvenido al sistema NeoGenesis. Esta guía te ayudará a navegar por la interfaz y sacar el máximo provecho a tus generaciones.

## 1. Selección de Modo
Al iniciar, verás 3 interruptores principales en la parte superior:
*   **QUICK vs ADVANCED:** Elige entre un asistente paso a paso o el panel de control completo.
*   **IMAGE vs VIDEO:** Cambia la estructura del prompt. "Video" prioriza acciones continuas sobre poses estáticas.
*   **MIDJOURNEY vs GENERIC:**
    *   *Midjourney:* Añade parámetros técnicos como `--ar 16:9 --v 6.0`.
    *   *Generic:* Usa lenguaje natural ("Cinematic lighting, 8k") compatible con DALL-E 3 o Stable Diffusion.

## 2. El Agente de Élite (Elite Agent)
Situado en el encabezado, este botón es tu "botón de pánico" para calidad instantánea.
*   Al pulsarlo, el sistema cambia a **Quick Mode**.
*   Genera un personaje aleatorio pero con una configuración profesional fija: **Formato Móvil (9:16)**, **Fondo Blanco** y **Cuerpo Entero**.
*   Ideal para crear rápidamente assets limpios para redes sociales (TikTok/Reels/Shorts).

## 3. Genome Chaos
El botón morado en el encabezado.
*   Al pulsarlo, el sistema cambia a **Advanced Mode**.
*   Aleatoriza **TODO**. Colores de ojos, cicatrices, accesorios, tipos de ropa...
*   Úsalo cuando necesites ideas locas o "mutaciones" creativas.

## 4. Opciones del Wizard (Quick Mode)
En el paso de "Fondo / Entorno", encontrarás opciones simplificadas:
*   **Escenario:** Genera un fondo detallado y complejo acorde al rol del personaje.
*   **Simple / Desenfocado:** Fondo borroso (Bokeh) para resaltar al personaje.
*   **Fondo Blanco:** Ideal para recortar el personaje después en Photoshop.

## 5. Botones de Generación

*   **GENERATE IMAGE:** Crea un único prompt detallado y optimizado.
*   **GENERATE INVENTORY:** Crea un prompt para una imagen de tipo "Knolling" (objetos organizados sobre fondo blanco) con el equipo de tu personaje.
*   **PSYCHE PROTOCOL (SHEETS):** Genera 7 prompts distintos para crear una hoja de personaje completa (Vistas técnicas, Tokens, Expresiones).
*   **NARRATIVE ARCHIVES:** Genera una ficha de "Lore" (Historia) con nombre, biografía y psicología. Los datos se muestran en pantalla listos para copiar y pegar.

## 6. Herramientas Útiles
*   **CRT_FX:** Botón superior derecho. Activa/Desactiva el efecto de monitor retro.
*   **MEMORY:** Abre el historial lateral para recuperar prompts anteriores.
*   **Footer:** En la parte inferior encontrarás enlaces directos a las redes de Norberto Cuartero y contacto.
